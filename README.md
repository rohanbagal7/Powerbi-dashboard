# Dashboard Features

The dashboard includes the following visualizations:

Line Chart – Monthly Sales Trend

Bar/Column Chart – Sales by Category & Region

 Donut Chart – Contribution by Product Category

 Slicers – Year, Quarter, Rank, Category , Product

kpi cards - Total sales, orders , profit

Matrix Table - Product & Region Details

Cards showing profit Margin and average order value
